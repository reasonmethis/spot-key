# Spot Key

A small floating pie-chart overlay for Windows that triggers keyboard shortcuts on hover.

## Development

- **Package manager:** uv
- **Run:** `uv run python -m spot_key`
- **Test:** `uv run pytest test_spot_key.py -v`
- **Visual test:** `uv run python visual_test.py` (renders pie on white/dark backgrounds, saves PNGs for edge inspection)

## Commit style

Include detailed, user/dev-friendly descriptions in every commit message. The description should be clear enough that someone who hasn't seen the code can understand what the commit does and why. Use a short title line plus a multi-line body explaining: what changed, why, and any notable implementation details.
